    Logiciels

    J'ai bien envie de vous donner un conseil pour les logiciels pour le travail de la création de votre site. Cela vous donnera les idées de vos choix.

    Voici les logiciels que je utilise actuellement sur mes ordinateurs pour le travail de la création des sites :

    Windows

    • Visual Studio Code - Un éditeur de code redéfini et optimisé pour créer et déboguer des applications web et cloud modernes - Gratuit : code.visualstudio.com
    • Cmder - Un progiciel qui offre une excellente expérience de console même sous Windows - Gratuit : cmder.net
    • FileZilla - La solution FTP pour le client et le serveur - Gratuit : filezilla-project.org
    • Github Desktop - Interface facile de la gestionnaire des projets via le serveur de Github sans l'utilisation de l'invite de commande Git - Gratuit : desktop.github.com
    • GitKraken - Interface facile de la gestionnaire des projets via les serveurs Git sans l'utilisation de l'invite de commande Git - Gratuit pour les utilisations personnelles et non-commercial : gitkraken.com
    • Pandoc : Je travaille beaucoup Markdown mais sur Windows, ce n'est vraiment pas top alors je suis obligé convertir de documents numériques via l'invite de commande - Gratuit : pandoc.org
    • Photoshop - Un logiciel de retouche, de traitement et de dessin assisté par ordinateur - Payant, 11 CHF par mois pour Photographie : adobe.com/photoshop
    • ImageMagick - Je l'utilise avec l'invite de commande pour de convertir, de modifier et d'afficher des images dans un très grand nombre de formats sans besoin de ouvrir un logiciel - Gratuit : imagemagick.org
    • JPEGmini - Le logiciel réduit la taille de l'image jusqu'à 80% sans compromettre la qualité - Payant 31.74 CHF en un seul crash : jpegmini.com

    macOS

    • Espresso - Un éditeur de code très intéressant et complet pour les sites statiques sur macOS - Payant : espressoapp.com
    • Atom.io - Un éditeur de code très complexe et complet - Gratuit : atom.io
    • Terminal - L'invite de commande sur macOS - Gratuit : Il est déjà inclus sur macOS.
    • Transmit - Un logiciel de FTP complet sur Mac - Payant : panic.com/transmit
    • Github Desktop - Interface facile de la gestionnaire des projets via le serveur de Github sans l'utilisation de l'invite de commande Git - Gratuit : desktop.github.com
    • GitKraken - Interface facile de la gestionnaire des projets via les serveurs Git sans l'utilisation de l'invite de commande Git - Gratuit pour les utilisations personnelles et non-commercial : gitkraken.com
    • CodeKit - Le logiciel compile automatiquement toutes les langues impressionnantes que vous avez lues dans les didacticiels - Payant 35 dollars en un seul crash : codekitapp.com
    • iA Writer - Un traitement en Markdown simple et complet quel j'adore travailler - Payant via Mac App Store en un seul crash : ia.net/writer
    • Affinity Photo - Un logiciel de retouche, de traitement et de dessin assisté par ordinateur - Payant pour 48 CHF, un seul crash : affinity.serif.com/photo
    • GraphicConverter - Un logiciel qui permet de lire des images à différents formats, les classer et de les convertir quel j'adore travailler depuis les longs années - Payant 39.95 euros en un seul crash via Mac App Store : lemkesoft.de/graphicconverter
    • JPEGmini - Le logiciel réduit la taille de l'image jusqu'à 80% sans compromettre la qualité - Payant via Mac App Store en un seul crash : jpegmini.com

    Voilà, c'est les logiciels quel je travaille pour créer les sites statiques sur Windows et sur macOS.

    Si vous ne connaîtrez rien des langages de HTML et CSS alors je vous donne un bon logiciel quel je utilisais après d'avoir utiliser iWeb (un logiciel qui est abandonné depuis des années par Apple) : Rapidweaver (payant) sur Mac ! Il est diponible en Français et facile à utiliser. Il fait tout pour toi.

    Mais si vous êtes sur Windows, C'est Website X5. Je conseille acheter un fois car c'est beaucoup moins cher qu'utiliser les serivces en ligne comme Wix ou Jimbo. En plus, c'est plus avantage d'utiliser un vrai hébergement.

    Gestion de contenu web

    Si vous avez prévu un projet de créer un très gros site professionnel ou vous modifiez ou créez beaucoup des pages et les articles, je vous conseille de contacter aux entreprises qui offre un CMS comme TYPO3 ou Drupal ou encore bien avec un thème à vos rêves à realiser. Et enfin, Wordpress est aussi très intéressant mais si c'est les sites professionnelles, je vous déconseille de ce CMS. Mon cas, je crée toujours aux sites sous Worpdress pour les association à but non lucratif, c'est un bon choix.

    Je vous présente quelques sites de ma région d'enfance qui utilisent quel système du CMS :

    • Visions du Réel, un festival internalional de cinéma Nyon : TYPO3
    • La Côte, Site des articles du journal et toute l'actualité en continu de la région de Nyon, Rolle, Aubonne, Morges, de Suisse et du Monde : Drupal
    • Design et Formation, il offre les formations intéressants du design : Wordpress
    • Paléo Festival Nyon : Un festival nyonnaise des musiques assez connu : Drupal
    • SINyon, un site de l'éngerie pour Nyon : Wordpress
    • Caribana Festival est un festival connu de la région nyonnais : Wordpress

    Merci de lire ce petit document qui vous apportera un réflexion de créer un site pour vous.

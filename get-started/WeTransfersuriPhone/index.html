    WeTransfer sur iPhone

    WeTransfer est un service de transfert de fichier fondé sur le cloud à Amsterdam et il existe depuis en 2009. On peut envoyer gratuitement aux fichiers volumineux jusqu'à 2 go par envoi avec le limite de stockage pendant environ 14 jours.

    Je donne un explication pour ceux qui n'arrivent pas m'envoyer les vidéos. Je vous prie de suivre les étapes à suivre :

    Aller sur App Store

    App Store : Télécharger WeTransfer

    Ouvrir Wetransfer

    Cliquer sur le bouton "+"

    Cliquez "Photos or Vidéo"

    Cela peut parfois attendre un temps de changement

    Cliquez "Create new board"

    Remplissez le formulaire

    $ Il est suffit que vous remplissez :

    • Name : Vous pouvez le laisser par un nom à votre idée.
    • Description (opional) : Vous pouvez exprimer par écrit ce que vous voulez.

    Et puis vous cliquez par Create

    Partagez

    Il est suffit de cliquer le bouton bleu Share

    Choix les options à partager

    Vous pouvez choisir ce que vous voulez :

    • Whatsapp ou iMessage
    • SMS
    • Email
    • Copier le lien - Il vous suffit cliquer Tap to copy link et puis coller sur un message ou un texte.
    • Autre chose possible.

    C'est tout ! Merci pour le lecture.
